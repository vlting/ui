# A11y: Composed Components — Accordion, Collapsible, Card, AlertDialog, Tooltip

## Audit Findings

### Accordion
- `style={{ all: 'unset' }}` on trigger `<button>` removes focus ring.
- No `aria-controls` or `id` attributes for association.

### Collapsible
- Trigger uses `cloneElement` to inject `onClick` and `aria-expanded` but no keyboard handlers.
- No `role`, no `aria-controls`, no `id` on content.

### Card (interactive)
- Clickable but no `tabIndex`, `role`, keyboard handling, or `focusStyle`.
- `CardTitle` renders as `<span>` not heading.

### AlertDialog
- Uses `role="dialog"` instead of required `role="alertdialog"`.

### Tooltip
- No `aria-describedby` linking trigger to tooltip. No `id` on tooltip content.
- No Escape key dismissal. Tooltip removed from DOM when hidden.

## Required Changes

### 1. Accordion — fix focus ring
- Replace `style={{ all: 'unset' }}` with minimal reset
- Add `focusWithinStyle` to `TriggerFrame` (outline ring)
- Generate IDs with `useId()` for `aria-controls`/`aria-labelledby` association
- Add `aria-controls={contentId}` to trigger, `id={contentId}` + `role="region"` + `aria-labelledby={triggerId}` to content

### 2. Collapsible — inject keyboard support
- Inject `onKeyDown` (Enter/Space), `role="button"`, `tabIndex={0}` via `cloneElement`
- Add `aria-controls` on trigger, `id` + `role="region"` on content

### 3. Card interactive — keyboard support
- Add `focusStyle` to `interactive: true` variant
- Add `tabIndex={0}`, `role="button"`, `onKeyDown` (Enter/Space) when interactive
- Add `tag: 'h3'` to `CardTitle`

### 4. AlertDialog — fix role
- Add `role` prop to headless Dialog Content, defaulting to `"dialog"`
- In styled AlertDialog, pass `role="alertdialog"`

### 5. Tooltip — aria-describedby and Escape
- Generate `id` with `useId()`, add to tooltip content
- Add `aria-describedby` to trigger wrapper
- Keep tooltip in DOM when hidden (`display: 'none'`) instead of unmounting
- Add Escape key handler on trigger to dismiss tooltip

## Verification

- Build: `yarn build`
- Keyboard: Tab to accordion triggers, cards, collapsible triggers. All show focus ring. Enter/Space activates.
- Screen reader: Accordion announces expanded/collapsed. Tooltip text announced on focus. AlertDialog announces as alertdialog.
- Escape: Tooltip dismisses on Escape.
