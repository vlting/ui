# A11y: Interactive Controls — Toggle, Slider, Switch, Checkbox, RadioGroup

## Priority: CRITICAL

- `packages/components/Checkbox/Checkbox.tsx`
- `packages/components/RadioGroup/RadioGroup.tsx`

## Audit Findings

### Toggle / ToggleGroup
- **Renders as `<div>`** — `ToggleFrame` and `ToggleGroupItemFrame` are `styled(XStack, ...)`. Not keyboard-focusable.
- Has `role="button"`, `aria-pressed` — but useless on a `<div>` without `tabIndex` and keyboard handlers.
- Focus styles exist but unreachable.

### Slider
- **No keyboard support** — Has `role="slider"` but no `tabIndex`, no arrow/Home/End handlers.
- **Thumb cannot receive focus** — `focusStyle` is defined but never activates.

### Switch
- **`style={{ all: 'unset' }}` on `<button>` removes focus ring** — No replacement focus style.
- **No hover style** on track.

### Checkbox
- **No visible focus or hover styles** on `StyledRoot`.
- Headless implementation is good (`<button>`, `role="checkbox"`, Space key).

### RadioGroup
- **`style={{ all: 'unset' }}` removes focus ring**.
- **No arrow key navigation** between items (WAI-ARIA requires roving tabindex).
- **No hover style**.

## Required Changes

### 1. Toggle — add `tag: 'button'`
Add `tag: 'button'` to `ToggleFrame` and `ToggleGroupItemFrame`. This enables native keyboard focus and Enter/Space activation. Existing `focusStyle` and `hoverStyle` will work.

### 2. Slider — keyboard support
- Move `role="slider"` and `aria-value*` props to the thumb element
- Add `tabIndex={0}` to thumb
- Add `onKeyDown` handler: ArrowRight/Up (+step), ArrowLeft/Down (-step), Home (min), End (max), PageUp/PageDown (big step)
- Add `aria-label` prop support

### 3. Switch — fix focus ring
- Replace `style={{ all: 'unset' }}` with minimal reset (`background: 'none'`, `border: 'none'`, `padding: 0`)
- Add `focusWithinStyle` to `SwitchTrack` for visible focus ring
- Add `hoverStyle` to `SwitchTrack`

### 4. Checkbox — focus and hover styles
- Add `focusWithinStyle` to `StyledRoot` (outline ring when parent `<button>` focused)
- Add `hoverStyle` (border color change)

### 5. RadioGroup — focus ring and arrow navigation
- Replace `style={{ all: 'unset' }}` with minimal reset
- Add visible focus ring via `focusWithinStyle` on `StyledCircle`
- Implement roving tabindex: only selected item has `tabIndex={0}`, others `tabIndex={-1}`
- Add ArrowUp/Down/Left/Right handlers to move selection between items
- Add `hoverStyle`

## Verification

- Build: `yarn build`
- Keyboard: Tab to every control. Enter/Space activates toggles. Arrow keys work on Slider and RadioGroup.
- Focus: Every control shows visible focus ring.
- Hover: Every control shows visible hover state.
