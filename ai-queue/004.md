# Create Pagination Component

## Problem

There is no Pagination component in the design system. This is a basic UI component that any component library should have.

## Design Reference

Follow the shadcn/ui Pagination pattern: a horizontal row of page number buttons with previous/next navigation, ellipsis for truncated ranges, and optional first/last page buttons.

## Required Implementation

### 1. Create `packages/components/Pagination/Pagination.tsx`

Build a compound component following the existing patterns in the codebase (see Button, Tabs, or Table for reference).

**Sub-components:**
- `Pagination.Root` — Horizontal container (`XStack`) with gap and alignment
- `Pagination.Previous` — Button to go to previous page (disabled on first page)
- `Pagination.Next` — Button to go to next page (disabled on last page)
- `Pagination.Item` — Individual page number button (active state for current page)
- `Pagination.Ellipsis` — "..." indicator for skipped page ranges

**Props for `Pagination.Root`:**
```ts
interface PaginationProps {
  currentPage: number
  totalPages: number
  onPageChange: (page: number) => void
  siblingCount?: number  // pages shown around current page (default 1)
  size?: 'sm' | 'md' | 'lg'
}
```

**Visual design guidelines:**
- Page items should be square-ish buttons with border radius
- Current/active page: `backgroundColor: '$color10'`, `color: '$color1'` (solid, like Button solid variant — but fix the contrast issue; if task 001 is already done, follow that pattern)
- Non-active pages: `backgroundColor: 'transparent'`, `borderWidth: 1`, `borderColor: '$borderColor'`
- Hover: `backgroundColor: '$backgroundHover'`
- Disabled (prev on page 1, next on last page): `opacity: 0.5`, `cursor: 'not-allowed'`
- Previous/Next should use text labels ("Previous" / "Next") or chevron arrows
- Size variants should control button height and font size (matching Button component sizes)

**Page range algorithm:**
Given `currentPage`, `totalPages`, and `siblingCount`, compute which page numbers to show:
- Always show first and last page
- Show `siblingCount` pages on each side of current page
- Use ellipsis (`...`) when there are gaps
- Example with totalPages=10, currentPage=5, siblingCount=1: `[1, ..., 4, 5, 6, ..., 10]`
