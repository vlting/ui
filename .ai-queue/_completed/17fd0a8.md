# Dashboard + App Shell Blocks

## Instructions

Create two blocks: `DashboardBlock` (rewrite of Dashboard01) and `AppShellBlock` (consolidation of the 3 originals).

### DashboardBlock

**Variants:**

| Variant | Description |
|---------|------------|
| `analytics` | Metrics cards + chart + data table (based on Dashboard01) |
| `overview` | Simpler metrics-only dashboard with stat cards in a grid |

**Props:**
```typescript
interface DashboardBlockProps {
  variant: 'analytics' | 'overview'
  title?: string
  description?: string
  // Analytics variant
  metrics?: MetricCard[]
  chartConfig?: ChartConfig
  chartData?: Record<string, unknown>[]
  chartXAxisKey?: string
  tableData?: Record<string, unknown>[]
  tableColumns?: ColumnDef[]
  // Sidebar navigation
  sidebarGroups?: NavGroup[]
  sidebarHeader?: ReactNode
  sidebarFooter?: ReactNode
  children?: ReactNode
}

interface MetricCard {
  title: string
  value: string | number
  change?: string
  trend?: 'up' | 'down' | 'neutral'
  icon?: ReactNode
}
```

### AppShellBlock

**Variants:**

| Variant | Description | Based On |
|---------|------------|----------|
| `sidebar-layout` | Full app shell with sidebar navigation | AppShellResponsive |
| `tab-layout` | Bottom tab navigation (mobile-friendly) | MobileTabLayout |
| `split-pane` | Master-detail split view | MasterDetail |

**Props:**
```typescript
interface AppShellBlockProps {
  variant: 'sidebar-layout' | 'tab-layout' | 'split-pane'
  // Sidebar layout
  sidebarGroups?: NavGroup[]
  sidebarHeader?: ReactNode
  sidebarFooter?: ReactNode
  // Tab layout
  tabs?: TabItem[]
  activeTab?: string
  onTabChange?: (tab: string) => void
  // Split pane
  masterContent?: ReactNode
  detailContent?: ReactNode
  masterWidth?: number
  showDetail?: boolean
  // Common
  children?: ReactNode
}
```

### Implementation

1. Create `packages/blocks/dashboard/DashboardBlock.tsx`
   - Reference `Dashboard01.tsx` for the analytics variant
   - Overview variant: grid of MetricCard components using Card from @vlting/ui
   - Both variants optionally include a sidebar

2. Create `packages/blocks/app-shell/AppShellBlock.tsx`
   - Reference `packages/blocks/originals/AppShellResponsive.tsx` for sidebar-layout
   - Reference `packages/blocks/originals/MobileTabLayout.tsx` for tab-layout
   - Reference `packages/blocks/originals/MasterDetail.tsx` for split-pane
   - Use @vlting/ui Tabs, Sidebar, Sheet components

3. Create `packages/blocks/app-shell/index.ts`

### Patterns
- AnyFC casting for all Tamagui/UI components
- Use existing NavGroup types from sidebar/_shared.tsx
- Use Chart, DataTable, Card, Sidebar, Tabs, Sheet from @vlting/ui
- Responsive: use Tamagui media queries ($md, $gtMd) for layout breakpoints

## Scope
