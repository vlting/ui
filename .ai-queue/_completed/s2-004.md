# Task: Convert docs interactive tool components from Tailwind to STL

## Context
Replacing Tailwind utility classes with STL styling. See the STL tools and token mapping reference in task 002 for conversion patterns.

## Available STL Tools
- `styled('div', { display: 'flex', gap: '$4' })` from `@vlting/stl-react`
- Token refs: `'$4'` → space, `'$primary9'` → color, `'$background'` → semantic
- Conditions: `{ dark: {...} }`, `{ md: {...} }`, `{ ':hover': {...} }`
