## Instructions

Create `PieChart.tsx` — Pie/donut chart component using custom SVG rendering via `react-native-svg`:

### Props
```tsx
export interface PieChartProps {
  data: ChartDataPoint[]
  variant?: PieChartVariant
  xAxisKey?: string
  showLabels?: boolean
  showTooltip?: boolean
  interactive?: boolean
  showLegend?: boolean
  tooltipVariant?: TooltipVariant
  innerRadius?: number       // 0 = full pie, >0 = donut
  outerRadius?: number
  padAngle?: number          // gap between slices in radians
  cornerRadius?: number
  startAngle?: number        // default 0
  endAngle?: number          // default 2*PI
  activeIndex?: number | null
  customLabel?: React.ComponentType<any>
  labelPosition?: 'inside' | 'outside'
}
```

### 11 Variants
1. **default** — basic pie chart with colored slices
2. **donut** — pie with innerRadius (default ~60% of outerRadius)
3. **donut-text** — donut with center text (total value or custom label)
4. **donut-active** — donut with active slice that expands on hover/press
5. **label** — pie with labels on each slice (percentage or value)
6. **custom-label** — pie with customLabel component for each slice
7. **label-list** — labels positioned outside the chart connected by lines
8. **legend** — pie with ChartLegend below
9. **separator** — slices with visible gaps (padAngle > 0)
10. **interactive** — hover/press highlights a slice, shows tooltip
11. **stacked** — nested pies (multiple rings from different data series)

### Implementation Pattern

Use `react-native-svg` directly (like AreaChart does):
```tsx
import { Svg, Path, G, Text as SvgText, Line } from 'react-native-svg'
import { useChartContext } from './Chart'
import { ChartLegend } from './ChartLegend'
import { ChartTooltip } from './ChartTooltip'
import type { ChartDataPoint, PieChartVariant, TooltipVariant } from './types'
```

**Core math for pie slices:**
- Total = sum of all data values
- Each slice angle = (value / total) * 2 * Math.PI
- Convert angles to SVG arc paths using:
  ```
  x = cx + r * Math.cos(angle)
  y = cy + r * Math.sin(angle)
  ```
- Use SVG arc command: `A rx ry x-rotation large-arc-flag sweep-flag x y`
- For donut: draw outer arc, then inner arc in reverse direction

**Key functions to implement:**
- `describeArc(cx, cy, outerR, innerR, startAngle, endAngle)` → SVG path string
- `getSliceData(data, xAxisKey, config)` → array of { key, value, percentage, startAngle, endAngle, color }

**For center text (donut-text):** Render `<SvgText>` at center (cx, cy) with total or custom text.

**For labels (label, label-list):**
- `label`: Position text at midpoint of arc (inside slice)
- `label-list`: Position text outside arc, draw connecting line from slice midpoint to label

**For interactive:** Track `activeIndex` state. Active slice: increase outerRadius by ~10px. Show ChartTooltip with slice data.

**For stacked (nested pies):** Iterate over config keys, each gets a ring at a different radius. Innermost ring smallest, outermost largest.
