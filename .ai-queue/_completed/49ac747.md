# Segment 2: Root Layout + Provider + Brand Switcher + Theme Toggle

Build the root layout with @vlting/ui Provider integration, brand switching, and dark/light theme toggle.

## Instructions

1. Create `examples/docs/app/layout.tsx`:
   - Root layout component
   - Import and apply `globals.css`
   - Wrap children in a `ThemeProvider` from `next-themes` (attribute="class", defaultTheme="system", enableSystem)
   - Wrap in `Providers` component (see step 2)
   - HTML lang="en", suppress hydration warning
   - Include Google Fonts preload links for Inter (default brand font)

2. Create `examples/docs/components/providers.tsx`:
   - Client component ("use client")
   - Uses React state to track selected brand (default, shadcn, fun, posh)
   - Creates Tamagui config via `createBrandConfig(selectedBrand)` + `createTamagui()`
   - Wraps children in `@vlting/ui` Provider with the active brand config
   - Exposes brand selection via React context (`BrandContext`)
   - Maps `next-themes` theme to Tamagui's `defaultTheme` prop (light/dark)

3. Create `examples/docs/components/brand-switcher.tsx`:
   - Client component ("use client")
   - Dropdown or segmented control showing brand options: Default, Shadcn, Fun, Posh
   - Reads/writes from BrandContext
   - Persists selection to localStorage

4. Create `examples/docs/components/theme-toggle.tsx`:
   - Client component ("use client")
   - Button that toggles between light/dark/system themes
   - Uses `useTheme()` from next-themes
   - Shows sun/moon icon based on current theme

5. Create `examples/docs/components/site-header.tsx`:
   - Header bar with site title "@vlting/ui", brand switcher, theme toggle
   - Sticky at top, border bottom
   - Link to GitHub repo

## Key Patterns
- Use `createTamagui` from `tamagui` package
- Use `createBrandConfig` and brand definitions (defaultBrand, shadcnBrand, funBrand, poshBrand) from `@vlting/ui`
- Provider from `@vlting/ui` wraps the Tamagui provider
- Must handle SSR â€” brands cannot be loaded dynamically during SSR (use default brand for SSR, hydrate with selected brand on client)
