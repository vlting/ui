## Instructions

### 1. Add FOUC-prevention script to `index.html`

In `examples/kitchen-sink/index.html`, add a blocking script BEFORE the closing `</head>` tag (but after the `<style>` block):

```html
<script>
  (function() {
    var m = localStorage.getItem('stl-color-mode');
    if (m) document.documentElement.setAttribute('data-color-mode', m);
  })()
</script>
```
