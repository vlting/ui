# Create Pagination Component

- Page range algorithm: always shows first/last, siblings around current, ellipsis for gaps

- ARIA labels (`aria-current="page"`, `aria-label`, `role="navigation"`)
- Active page: solid background (`$color10`) with contrast text (`$color1`)
- Non-active: transparent with border, hover state
- Added barrel exports in `packages/components/index.ts` and `src/index.ts`
