### `usePopoverPosition.ts`
Position an overlay relative to a trigger element.
```ts
interface UsePopoverPositionProps {
  placement?: 'top' | 'bottom' | 'left' | 'right' | 'top-start' | 'top-end' | 'bottom-start' | 'bottom-end'
  offset?: number
  triggerRef: RefObject<HTMLElement>
  contentRef: RefObject<HTMLElement>
  isOpen: boolean
}
interface UsePopoverPositionReturn {
  position: { top: number; left: number }
  actualPlacement: string  // may flip if near viewport edge
  update: () => void
}
```
- Use `getBoundingClientRect()` for positioning
- Flip placement if content would overflow viewport
- Return a manual `update()` for resize/scroll handlers

### `useTabs.ts`
Tab state with keyboard navigation.
```ts
interface UseTabsProps {
  defaultValue?: string

  onValueChange?: (value: string) => void
  orientation?: 'horizontal' | 'vertical'
}
interface UseTabsReturn {
  activeValue: string
  setActiveValue: (v: string) => void

}
```
- Arrow Left/Right (horizontal) or Up/Down (vertical) navigation
- Home/End for first/last tab
- Controlled/uncontrolled support

### `useContextMenu.ts`
