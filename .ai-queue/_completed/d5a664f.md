### 14 Variants
1. **default** — basic radar with polygon grid, filled area
2. **dots** — radar with visible dot markers at each vertex
3. **lines** — radar outline only (no fill, just stroked path)
4. **custom** — radar with custom styling (adjusted opacity, stroke)
5. **multiple** — multiple overlapping radar series from config
6. **grid-circle** — circular grid lines instead of polygon
7. **grid-filled** — alternating filled grid rings (zebra pattern)
8. **grid-none** — no grid lines, just the data shape
9. **legend** — radar with ChartLegend
10. **icons** — icons from config in tooltip/legend
11. **radius-axis** — shows value labels along one radius axis
12. **label** — shows category labels at each vertex
13. **custom-shape** — uses customShape component for vertex markers
14. **interactive** — hover/press on vertex shows tooltip

**Core math for radar chart:**
Radar charts use polar coordinates with equal angular spacing:
- N categories = N vertices equally spaced around circle
- Angle per category = 2 * Math.PI / N
- For each data point: radius = (value / maxValue) * outerRadius
- Vertex position: x = cx + r * Math.sin(angle), y = cy - r * Math.cos(angle)
  (Note: sin for x, -cos for y to start from top)

**Key functions to implement:**
- `getVertexPosition(cx, cy, radius, angleIndex, totalAngles)` → {x, y}
- `buildRadarPath(data, categories, maxValue, outerRadius, cx, cy)` → SVG path string (closed polygon)
- `buildGridPath(level, totalLevels, categories, outerRadius, cx, cy, gridType)` → SVG path string

**Grid rendering:**
- `polygon` grid: connect vertices at each level with straight lines
- `circle` grid: draw concentric circles at each level
- `grid-filled`: fill alternating rings with subtle background color
- `grid-none`: skip grid entirely

**Multi-series (multiple):** For each key in config, build a separate radar path with its color and fillOpacity.

**Labels:** Position category names at each vertex, slightly outside the outerRadius.

**Radius axis:** Draw a line from center to one vertex with tick marks and value labels.

**Interactive:** Track hovered vertex index. Show ChartTooltip with the values at that vertex across all series.
