## Instructions

### 1. Sidebar05 — Collapsible Submenus

Similar to Sidebar03 but with a different collapse animation/behavior. Submenus slide open with animation. Each submenu parent shows a chevron that rotates when expanded.

Layout:
- `Sidebar.Root` standard variant
- Groups with collapsible submenu items
- The key difference from Sidebar03: uses a smoother transition pattern with `Collapsible` and animated chevron rotation
- Each parent item with children has: icon (optional) + label + animated chevron

Props: `Sidebar05Props extends SidebarBlockProps`

### 2. Sidebar06 — Submenus as Dropdowns

Instead of inline collapsible submenus, child navigation items appear in dropdown menus triggered by hovering or clicking the parent item.

Layout:
- `Sidebar.Root` standard variant
- Top-level items render normally
- Items with `children` use `DropdownMenu` (from `packages/components/DropdownMenu`):
  - `DropdownMenu.Root` wrapping the parent item
  - `DropdownMenu.Trigger` on the parent MenuItem
  - `DropdownMenu.Content` containing child items as `DropdownMenu.Item`
- NOTE: DropdownMenu may have Tamagui v2 RC typing issues — use AnyFC casts as needed

Props: `Sidebar06Props extends SidebarBlockProps`

### 3. Sidebar07 — Collapses to Icons

A sidebar that uses `collapsible="icon"` mode. When collapsed, only icons are visible. When expanded, full labels appear.

Layout:
- `Sidebar.Root collapsible="icon"` with `Sidebar.Trigger`
- When expanded: normal sidebar with icon + label for each item
- When collapsed: only icons visible (48px width default)
- `Sidebar.GroupLabel` auto-hides in collapsed mode (built-in behavior)
- Items should show Tooltip with label text when sidebar is collapsed
- Requires each NavItem to have an `icon` — items without icons should not be rendered in collapsed mode

Props: `Sidebar07Props extends SidebarBlockProps` with optional `showTrigger?: boolean`

### 4. Spec and test files

Specs: document the specific submenu/collapse behavior for each variant.

Tests should verify:
- Sidebar05: submenu toggle behavior, nested items render
- Sidebar06: dropdown trigger renders, child items in dropdown
- Sidebar07: icon-collapse mode, trigger toggles sidebar width, icons visible when collapsed
- All: navigation items render, active states, landmarks
