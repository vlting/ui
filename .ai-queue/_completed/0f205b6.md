# Polish Switch, Checkbox, and Calendar visual quality

## Instructions

### Context
These three components are functional but lack visual polish. Each needs refined styling to match shadcn-level quality.

Read `DESIGN_CONSTITUTION.md` before making changes.

### Switch Improvements (`packages/components/Switch/Switch.tsx`)

1. **Track appearance**: Improve the off-state track color — use `$color4` for off state and `$color10` for on state with clear visual differentiation.

2. **Thumb shadow**: Add a subtle shadow to the thumb for depth: `shadowColor: '$shadowColor'`, `shadowRadius: 1`, `shadowOffset: { width: 0, height: 1 }`.

3. **Size-based padding**: Replace hardcoded `padding: 3` with size-proportional padding using tokens.

4. **Transition polish**: Ensure smooth transition between on/off states.

### Checkbox Improvements (`packages/components/Checkbox/Checkbox.tsx`)

1. **Replace Unicode checkmark**: Replace the '✓' unicode character with a proper SVG checkmark. Use a simple inline SVG path or import the Check icon from `@tamagui/lucide-icons`.

2. **Border weight**: Increase to 2px border for better visibility.

3. **Checked state**: Use `$color10` background with white/contrast check indicator.

4. **Hover state**: Add `hoverStyle` with subtle border color change (e.g., `borderColor: '$color8'`).

5. **Indeterminate state**: Add a horizontal dash (—) indicator for the indeterminate state, styled similarly to the check indicator.

6. **Label spacing**: Replace hardcoded `gap: 8` with token-based spacing (e.g., `gap: '$2'`).

### Calendar Improvements (`packages/components/Calendar/Calendar.tsx`)

1. **Navigation arrows**: Replace '<' and '>' text with proper chevron icons. Import ChevronLeft/ChevronRight from `@tamagui/lucide-icons` or use SVG paths.

2. **Day cell hover**: Add `hoverStyle` with subtle background (e.g., `backgroundColor: '$color3'`).

3. **Today indicator**: Add a distinct visual for today — a subtle ring or background tint (e.g., `borderColor: '$color8'`, `borderWidth: 1`).

4. **Cell spacing**: Add small gap between day cells for visual breathing room.

5. **Out-of-month dates**: Show placeholder day cells for days from adjacent months with very muted text (`color: '$color5'`, `opacity: 0.4`).

6. **Header styling**: Add subtle bottom border to separate month/year header from the day grid.

## Scope

- `packages/components/Checkbox/Checkbox.tsx`
- `packages/components/Calendar/Calendar.tsx`
