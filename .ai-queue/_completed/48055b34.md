# Task: Create root vite.config.ts for library build

## Context

- `"main": "dist/cjs/index.cjs"`
- `"module": "dist/esm/index.mjs"`
- `"types": "dist/types/index.d.ts"`

The build needs to produce ESM and CJS bundles from `src/index.ts`.

Reference: Look at `packages/stl/vite.config.ts` or `packages/stl-react/vite.config.ts` for patterns already used in this repo.

1. Create `vite.config.ts` at repo root with library mode:
   - Entry: `src/index.ts`
   - Output formats: `es` (→ `dist/esm/index.mjs`) and `cjs` (→ `dist/cjs/index.cjs`)
   - External all dependencies: `react`, `react-dom`, `react-native`, plus all `@vlting/*` sub-packages

   - Enable source maps
   - Set `build.outDir` to `dist`

     },
     rollupOptions: {
       external: [/^react/, /^@vlting\//, /node_modules/],
