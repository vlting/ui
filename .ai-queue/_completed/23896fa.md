### Component API Reference

Use these patterns for each component type. ALL components are imported from `@vlting/ui`.

**Simple components** (single element, no sub-components):
- `Badge`: `<Badge variant="default">New</Badge>` — variants: default, secondary, destructive, outline, subtle
- `Separator`: `<Separator orientation="horizontal" />`
- `Skeleton`: `<Skeleton width={200} height={20} />` + `<Skeleton circle width={40} height={40} />`
- `Spinner`: `<Spinner size="md" />`
- `Kbd`: `<Kbd>⌘</Kbd> + <Kbd>K</Kbd>`
- `Label`: `<Label htmlFor="email" required>Email</Label>`
- `Switch`: `<Switch checked={true} size="md" />`
- `Progress`: `<Progress value={75} max={100} />`
- `Slider`: `<Slider defaultValue={50} min={0} max={100} />`
- `Textarea`: `<Textarea label="Comments" placeholder="Enter text..." rows={4} />`
- `Toggle`: `<Toggle pressed={true} variant="outline">Bold</Toggle>`
- `Loader`: `<Loader size="md" />`
- `AspectRatio`: Container with `<AspectRatio ratio={16/9}>` wrapping content

**Compound components** (Root + sub-components):
- `Alert`: `Alert.Root` > `Alert.Title` + `Alert.Description` — variant: 'default' | 'destructive'
- `Avatar`: `Avatar` > `Avatar.Image` + `Avatar.Fallback` — size: 'sm'|'md'|'lg'|'xl'

- `Empty`: `Empty.Root` > `Empty.Title` + `Empty.Description` + `Empty.Action`
- `Collapsible`: `Collapsible.Root` > `Collapsible.Trigger` + `Collapsible.Content`
- `Popover`: `Popover.Root` > `Popover.Trigger` + `Popover.Content`

- `Drawer`: `Drawer.Root` > `Drawer.Trigger` + `Drawer.Content` > `Drawer.Header` + `Drawer.Title`
- `AlertDialog`: `AlertDialog.Root` > `AlertDialog.Trigger` + `AlertDialog.Overlay` > `AlertDialog.Content` > `AlertDialog.Title` + `AlertDialog.Description` + `AlertDialog.Footer` > `AlertDialog.Cancel` + `AlertDialog.Action`
- `HoverCard`: `HoverCard.Root` > `HoverCard.Trigger` + `HoverCard.Content`
