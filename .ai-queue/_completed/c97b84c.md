# Create shadcn brand definition for pixel-perfect shadcn/ui match

## Goal

Create a new `shadcn` brand config that, when active, makes our components visually identical to shadcn/ui's default "Neutral" theme. This is the foundation for pixel-perfect parity.

## shadcn/ui Design Tokens (source of truth)

shadcn/ui uses OKLCH colors with a semantic naming convention. Here are the exact values:

### Light mode
```
--radius: 0.625rem
--background: oklch(1 0 0)                          // #ffffff
--foreground: oklch(0.145 0 0)                       // ~#0a0a0a
--card: oklch(1 0 0) / card-foreground: oklch(0.145 0 0)
--popover: oklch(1 0 0) / popover-foreground: oklch(0.145 0 0)
--primary: oklch(0.205 0 0) / primary-foreground: oklch(0.985 0 0)    // near-black / near-white
--secondary: oklch(0.97 0 0) / secondary-foreground: oklch(0.205 0 0) // light gray / near-black
--muted: oklch(0.97 0 0) / muted-foreground: oklch(0.556 0 0)         // light gray / medium gray
--accent: oklch(0.97 0 0) / accent-foreground: oklch(0.205 0 0)
--destructive: oklch(0.577 0.245 27.325)                               // red
--border: oklch(0.922 0 0)                            // ~#e5e5e5
--input: oklch(0.922 0 0)
--ring: oklch(0.708 0 0)
```

### Dark mode
```
--background: oklch(0.145 0 0)                        // ~#0a0a0a
--foreground: oklch(0.985 0 0)                        // ~#fafafa
--card: oklch(0.205 0 0) / card-foreground: oklch(0.985 0 0)
--primary: oklch(0.922 0 0) / primary-foreground: oklch(0.205 0 0)    // inverted
--secondary: oklch(0.269 0 0) / secondary-foreground: oklch(0.985 0 0)
--muted: oklch(0.269 0 0) / muted-foreground: oklch(0.708 0 0)
--destructive: oklch(0.577 0.245 27.325) (same red)
--border: oklch(1 0 0 / 10%)                          // ~#ffffff1a
--input: oklch(1 0 0 / 15%)
--ring: oklch(0.556 0 0)
```

### Typography
- Font: Inter (or system font stack: -apple-system, system-ui, sans-serif)
- Base font size: 14px (0.875rem) for components, 16px for body
- Font weights: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)
- Letter spacing: tight for headings (-0.025em)

### Sizing / Spacing
- Border radius: 0.625rem (10px) base, with variants: sm=calc(var(--radius)-4px), md=calc(var(--radius)-2px), lg=var(--radius)
- Border width: 1px
- Button heights: default=36px (h-9), sm=32px (h-8), lg=40px (h-10)
- Input height: 36px (h-9)
- Padding: buttons px-4 py-2, inputs px-3 py-1

```ts
import type { BrandDefinition } from './index'

// Convert OKLCH to hex for our palette system
// Light palette: 12 steps from lightest to darkest (matching our template mapping)
// Step 0  → background (#ffffff)
// Step 1  → card surface (#ffffff)
// Step 2  → secondary/muted (#f5f5f5, oklch 0.97)
// Step 3  → border/input (#e5e5e5, oklch 0.922)
// Step 4  → ~#d4d4d4 (interpolated)
// Step 5  → ~#c4c4c4 (interpolated)
// Step 6  → ~#a3a3a3 (interpolated)
// Step 7  → muted-foreground (#737373, oklch 0.556)
// Step 8  → ring (#525252, oklch 0.708 → actually this might be ~#b4b4b4)
// Step 9  → ~#404040 (interpolated)
// Step 10 → primary (#171717, oklch 0.205)
// Step 11 → foreground (#0a0a0a, oklch 0.145)

export const shadcnBrand: BrandDefinition = {
  name: 'shadcn',
  palettes: {
    light: [/* 12 hex values, lightest to darkest */],
    dark: [/* 12 hex values for dark mode, darkest to lightest (inverted) */],
  },
  accentPalettes: {
    red: {
      light: [/* 12-step red scale for destructive */],
      dark: [/* 12-step red scale for dark mode */],
    },
  },
  tokens: {
    radius: {
      // Map our radius tokens to shadcn's values
      // shadcn base --radius is 0.625rem (10px)
      // Our tokens: $1, $2, $3, $4, etc.
      // Map so $4 (commonly used) = 10px
    },
    size: {
      // Button/input heights
      // $3.5 = 32px (sm), $4 = 36px (default), $4.5 = 40px (lg)
    },
    space: {
      // Padding values
    },
  },
  borders: {
    widths: { thin: 1 },
  },
  outline: { width: 2, offset: 2 },
  shadows: {
    light: {
      // shadcn uses very subtle shadows
      sm: '0 1px 2px 0 rgb(0 0 0 / 0.05)',
      md: '0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1)',
      lg: '0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)',
    },
    dark: {
      // Similar but adapted for dark backgrounds
    },
  },
  fonts: {
    heading: {
      family: 'Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif',
      // Weight map matching shadcn
    },
    body: {
      family: 'Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif',
    },
  },
  typography: {
    heading: { letterSpacing: '-0.025em' },
  },
}
```

**IMPORTANT**: The agent must convert OKLCH values to hex accurately. Use JavaScript or an online converter to get exact hex values. The OKLCH values with 0 chroma are pure grays:
- oklch(1 0 0) = #ffffff
- oklch(0.985 0 0) ≈ #fafafa
- oklch(0.97 0 0) ≈ #f5f5f5
- oklch(0.922 0 0) ≈ #e5e5e5
- oklch(0.708 0 0) ≈ #b3b3b3
- oklch(0.556 0 0) ≈ #737373
- oklch(0.269 0 0) ≈ #262626
- oklch(0.205 0 0) ≈ #171717
- oklch(0.145 0 0) ≈ #0a0a0a

For the destructive oklch(0.577 0.245 27.325), convert to hex (approximately #ef4444 or similar red).
