### Sidebar13 — Sidebar in a Dialog
Navigation accessed through a Dialog (modal). The dialog contains a full sidebar-style navigation panel. Good for mobile patterns where navigation takes over the screen.

**Props:** `Sidebar13Props extends Omit<SidebarBlockProps, 'open' | 'onOpenChange' | 'collapsible' | 'variant' | 'side'>` plus:
- `triggerLabel?: string` (default: "Menu")
- `dialogTitle?: string` (default: "Navigation")
- `dialogOpen?: boolean`
- `onDialogOpenChange?: (open: boolean) => void`

**Layout:**
- Dialog.Root wrapping everything
- Dialog.Trigger: a Button with triggerLabel text
- Dialog.Content (size="lg"): Dialog.Header with Dialog.Title, then a `<nav>` element containing navigation groups
- Navigation groups rendered with SidebarNavGroup helper (or manually with Group/GroupLabel/Menu/MenuItem)
- Dialog.Close in header corner
- Uses `styledHtml('nav', {...})` for the nav landmark inside dialog

Create: `packages/blocks/sidebar/Sidebar13.tsx`, `packages/blocks/sidebar/Sidebar13.spec.md`, `packages/blocks/sidebar/Sidebar13.test.tsx`

### Sidebar14 — Sidebar on the Right
Standard sidebar but positioned on the right side. Demonstrates `side="right"` usage.

**Props:** `Sidebar14Props extends SidebarBlockProps` — same as standard sidebar but defaults `side` to "right"

**Layout:**
- Identical to Sidebar01 structurally (simple grouped navigation)
- Key difference: `side="right"` on Sidebar.Root
- Uses SidebarNavGroup helper for groups
- Includes Sidebar.Trigger positioned appropriately for right-side context

Create: `packages/blocks/sidebar/Sidebar14.tsx`, `packages/blocks/sidebar/Sidebar14.spec.md`, `packages/blocks/sidebar/Sidebar14.test.tsx`
