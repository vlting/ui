# Fix HoverCard positioning, Command search error, and Dialog title whitespace

## Instructions

### Problem 1: HoverCard renders at bottom of page
The HoverCard content renders at the bottom of the page instead of near the trigger element.

**Root Cause**: In `packages/components/HoverCard/HoverCard.tsx`, the `Root` component (lines 59-64) is missing `position="relative"`. It was removed in a previous commit but the `Content` component uses `position="absolute"` (line 105) and needs a positioned ancestor to render correctly near the trigger.

**Fix**: Add `position="relative"` back to the `Root` component's `ViewJsx`:
```tsx
// Line ~61 in HoverCard.tsx
<ViewJsx position="relative" display="inline-flex">{children}</ViewJsx>
```

### Problem 2: Command throws error when typing in search
The Command component throws an error when typing into the search field.

**Root Cause**: In `apps/docs/components/component-examples.tsx` (lines 862-864), the `Command.Item` components are missing the required `value` prop. The `CommandItemProps` interface in `Command.tsx` (line 32) requires `value: string`. When searching, the component tries to use `value` for text matching at line 157 (`searchableText = keywords ? [...keywords, value].join(' ') : value`) and gets `undefined`.

**Fix**: Add `value` props to all `Command.Item` components in the docs example:
```tsx
<Command.Item value="calendar">Calendar</Command.Item>
<Command.Item value="search">Search</Command.Item>
<Command.Item value="settings">Settings</Command.Item>
```

### Problem 3: Dialog title has excessive line-height
The Dialog title has excessive whitespace/line-height at the top.

**Root Cause**: In `packages/components/Dialog/Dialog.tsx` (lines 11-16), the `StyledTitle` styled component is missing a `lineHeight` token. Without it, Tamagui defaults to a line-height that's too tall for the `$6` font size.

**Fix**: Add `lineHeight: '$6'` to the `StyledTitle` styled component:
```tsx
const StyledTitle = styled(Text, {
  fontFamily: '$heading',
  fontWeight: '$4',
  fontSize: '$6',
  lineHeight: '$6',
  color: '$color',
})
```
