# RadialChart Component

Create the RadialChart with 6 variants using custom SVG rendering (react-native-svg).

## Scope
- `packages/components/Chart/RadialChart.tsx`

## Instructions

Create `RadialChart.tsx` — Radial bar/progress chart using custom SVG rendering via `react-native-svg`. A radial chart displays data as partial arcs around a center point, like a circular progress bar or gauge.

### Props
```tsx
export interface RadialChartProps {
  data: ChartDataPoint[]
  variant?: RadialChartVariant
  xAxisKey?: string
  showLabels?: boolean
  showGrid?: boolean
  showTooltip?: boolean
  showLegend?: boolean
  tooltipVariant?: TooltipVariant
  innerRadius?: number
  outerRadius?: number
  startAngle?: number        // default -90 (top of circle, in degrees)
  endAngle?: number          // default 270 (full circle from top, in degrees)
  barWidth?: number          // thickness of each arc (default 20)
  barGap?: number            // gap between concentric arcs (default 4)
  maxValue?: number          // explicit max for arc extent (default: auto from data)
  showText?: boolean         // center text
  centerText?: string        // custom center text
  customShape?: React.ComponentType<any>  // custom arc cap
}
```

### 6 Variants
1. **default** — single or multiple concentric arcs showing progress
2. **grid** — background track arcs visible behind data arcs
3. **label** — value labels at the end of each arc
4. **stacked** — arcs stacked end-to-end in a single ring (like stacked bar in circular form)
5. **text** — center text showing total or primary value
6. **shape** — custom end caps on arcs (rounded, square, or custom component)

### Implementation Pattern

Use `react-native-svg` directly:
```tsx
import { Svg, Path, G, Text as SvgText, Circle } from 'react-native-svg'
import { useChartContext } from './Chart'
import { ChartLegend } from './ChartLegend'
import type { ChartDataPoint, RadialChartVariant, TooltipVariant } from './types'
```

**Core math for radial arcs:**
Each data series gets its own concentric ring. The arc extent represents the data value relative to maxValue.

- Convert degrees to radians: `rad = deg * Math.PI / 180`
- Arc extent for a series: `arcAngle = (value / maxValue) * (endAngle - startAngle)`
- For multiple series: stack rings from outerRadius inward, each ring `barWidth` thick with `barGap` between

**Key functions to implement:**
- `describeArc(cx, cy, radius, startAngleDeg, endAngleDeg)` → SVG arc path string
- `getArcEndpoint(cx, cy, radius, angleDeg)` → {x, y}

**SVG arc path:**
```
M startX startY
A radius radius 0 largeArcFlag 1 endX endY
```
- largeArcFlag = 1 if arc > 180 degrees, else 0

**Grid variant:** Draw a full-circle background arc (track) in a muted color before the data arc.

**Label variant:** Position text at the endpoint of each arc.

**Stacked variant:** Instead of concentric rings, stack arcs end-to-end in a single ring. Each series starts where the previous one ended.

**Text variant:** Render `<SvgText>` at center with value or custom text.

**Shape variant:** Add decorative end caps using `strokeLinecap="round"` or custom SVG shapes at arc endpoints.
