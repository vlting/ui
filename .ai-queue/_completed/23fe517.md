## Instructions

### Migration Pattern

| Old pattern | New pattern |
|------------|-------------|
| `text-gray-600 dark:text-gray-400` | `text-foreground-secondary` |
| `text-gray-500 dark:text-gray-400` | `text-muted-foreground` |
| `text-gray-700 dark:text-gray-300` | `text-foreground-secondary` |
| `text-gray-400 dark:text-gray-600` | `text-muted-foreground` |
| `border-gray-200 dark:border-gray-800` | `border-border` |
| `border-gray-100 dark:border-gray-800/50` | `border-border-muted` |
| `border-gray-300 dark:border-gray-700` | `border-border` |
| `bg-gray-50 dark:bg-gray-900` | `bg-surface-muted` |
| `bg-white dark:bg-gray-950` | `bg-background` |
| `bg-white dark:bg-gray-800` (inputs) | `bg-surface` |
| `bg-gray-200 dark:bg-gray-800` | `bg-accent` |
| `hover:bg-gray-100 dark:hover:bg-gray-800` | `hover:bg-accent` |
| `hover:text-gray-900 dark:hover:text-gray-100` | `hover:text-foreground` |
