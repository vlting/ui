### 1. Create `packages/components/Chart/BarChart.tsx`

Define and export `BarChartVariant` type inline:
```typescript
export type BarChartVariant = 'default' | 'horizontal' | 'stacked' | 'grouped' | 'negative' | 'label' | 'mixed' | 'custom-label' | 'active' | 'interactive'
```

Define and export `BarChartProps`:
```typescript
export interface BarChartProps {
  data: ChartDataPoint[]
  variant?: BarChartVariant
  config: ChartConfig
  xAxisKey?: string
  showXAxis?: boolean
  showYAxis?: boolean
  showGrid?: boolean
  horizontal?: boolean
  stacked?: boolean
  grouped?: boolean
  showLabels?: boolean
  showTooltip?: boolean
  interactive?: boolean
  showLegend?: boolean
  tooltipVariant?: TooltipVariant
  barWidth?: number
  cornerRadius?: number
  activeBarIndex?: number | null
  customLabel?: React.ComponentType<any>
}
```

Import Victory components:
```typescript
import { VictoryChart, VictoryBar, VictoryStack, VictoryGroup, VictoryAxis, VictoryVoronoiContainer, VictoryLabel } from 'victory-native'
```

Implementation approach:
- Use `useChartContext()` to get `resolvedColors`, `victoryTheme`, `dimensions`
- Derive variant-specific behavior:
  - `variant === 'horizontal'` OR `horizontal === true`: Pass `horizontal` prop to `VictoryChart`
  - `variant === 'stacked'` OR `stacked === true`: Wrap bars in `<VictoryStack>`
  - `variant === 'grouped'` OR `grouped === true`: Wrap bars in `<VictoryGroup offset={barWidth + 4}>`
  - `variant === 'negative'`: No special prop — VictoryBar handles negative values natively. Ensure axis crosses at 0.
  - `variant === 'label'` OR `showLabels === true`: Add `labels={({ datum }: any) => datum.y}` and `labelComponent={<VictoryLabel />}` to VictoryBar
  - `variant === 'mixed'`: Color bars based on sign — positive gets series color, negative gets a muted variant
  - `variant === 'custom-label'`: Use `customLabel` prop as `labelComponent`
  - `variant === 'active'`: When `activeBarIndex` is set, apply highlight style to that bar index and dim others
  - `variant === 'interactive'` OR `interactive === true`: Use `VictoryVoronoiContainer`
- For multi-series: iterate over `Object.keys(config)` to render one `VictoryBar` per series
- Apply `resolvedColors` from context to each series' `style.data.fill`
- Default `barWidth` is undefined (let Victory auto-size)
- Apply `cornerRadius` to `VictoryBar`'s `cornerRadius` prop when set

**CRITICAL patterns:**
- `theme={victoryTheme as any}` — Victory's types are strict
- For horizontal charts, also swap X/Y axis labels if both axes are shown
- For grouped bars, the `offset` prop on VictoryGroup controls the gap between bars in each group
- For the `active` variant, use `style.data` function form: `fill: ({ index }: any) => index === activeBarIndex ? resolvedColors[key] : resolvedColors[key] + '40'`
