# Task: Migrate playground App.tsx to new styled() API

## Transform
Every `styled()` call changes from:
```ts
styled('tag', { stl: { ...styles }, styleName: 'Name' })
```
to:
```ts
styled('tag', { ...styles }, { name: 'Name' })
```

- `stl` object contents become the 2nd positional arg directly
- `styleName` → `name` in optional 3rd config arg
- No variant usage — all calls are simple
