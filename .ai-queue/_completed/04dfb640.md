# Task: Convert docs code display & preview components from Tailwind to STL

## Context
Replacing Tailwind utility classes with STL styling. See the STL tools and token mapping reference in task 002 for the conversion patterns.

## Available STL Tools
- `styled('div', { display: 'flex', gap: '$4' })` from `@vlting/stl-react`
- Token refs: `'$4'` → space, `'$primary9'` → color, `'$background'` → semantic
- Conditions: `{ dark: {...} }`, `{ md: {...} }`, `{ ':hover': {...} }`

## Approach
For each file:
1. Read the file to understand current Tailwind usage
2. Create styled components using `styled()` from `@vlting/stl-react`
3. Replace all `className="..."` patterns
4. Use STL color tokens for backgrounds, borders, text
5. Use STL space tokens for padding, gap, margin
