# A11y Fix: Toggle & ToggleGroup — Use Native `<button>` Elements

## Priority: HIGH

## Root Cause

`styled(XStack, { tag: 'button' })` renders `<div tag="button">`, not `<button>`. Both `ToggleFrame` and `ToggleGroupItemFrame` are not keyboard-focusable.

## Scope

Files modified:
- `packages/components/Toggle/Toggle.tsx`

## Required Changes

### 1. Toggle — wrap in native `<button>`

Same pattern as Button fix. Render a native `<button>` wrapping the styled visual frame:

```tsx
function Toggle({ pressed, onPressedChange, disabled, children, ...props }) {
  return (
    <button
      type="button"
      role="button"
      aria-pressed={pressed}
      aria-disabled={disabled || undefined}
      disabled={disabled}
      onClick={() => !disabled && onPressedChange?.(!pressed)}
      style={{ all: 'unset', display: 'inline-flex', cursor: disabled ? 'not-allowed' : 'pointer' }}
    >
      <ToggleFrameVisual pressed={pressed} disabled={disabled} {...props}>
        {children}
      </ToggleFrameVisual>
    </button>
  )
}
```

Remove `tag: 'button'` from `ToggleFrame` styled() call.

### 2. Add focusWithinStyle to ToggleFrameVisual

```tsx
focusWithinStyle: {
  outlineWidth: 2,
  outlineOffset: 2,
  outlineColor: '$outlineColor',
  outlineStyle: 'solid',
},
```

### 3. ToggleGroupItem — same treatment

Wrap `ToggleGroupItemFrame` in a native `<button>`:

```tsx
function ToggleGroupItem({ value, children }) {
  const { value: groupValue, onValueChange, type } = useContext(ToggleGroupContext)
  const isActive = type === 'single' ? groupValue === value : (groupValue as string[])?.includes(value)

  return (
    <button
      type="button"
      role="button"
      aria-pressed={isActive}
      onClick={() => onValueChange(value)}
      style={{ all: 'unset', display: 'inline-flex', cursor: 'pointer' }}
    >
      <ToggleGroupItemVisual isActive={isActive}>
        {children}
      </ToggleGroupItemVisual>
    </button>
  )
}
```

Remove `tag: 'button'` from `ToggleGroupItemFrame`.

### 4. Add focusWithinStyle to ToggleGroupItemVisual

Same pattern as Toggle.
