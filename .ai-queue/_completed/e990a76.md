## Instructions

### Test: createIcon factory
```tsx
import { render, screen } from '../../../src/__test-utils__/render'
import { createIcon } from '../createIcon'
```

Tests to write:

1. **renders an SVG element** ‚Äî call `createIcon('M0 0h24v24H0z', 'TestIcon')`, render the result, verify an `<svg>` element is present in the DOM.

2. **renders a path with correct d attribute** ‚Äî render a created icon, find the `<path>` element, verify its `d` attribute matches the path data passed to `createIcon`.

3. **forwards size prop** ‚Äî render with `size={32}`, verify the SVG has `width="32"` and `height="32"`.

4. **forwards color prop** ‚Äî render with `color="red"`, verify the SVG has `fill="red"`.

5. **defaults to size 24** ‚Äî render without size prop, verify SVG has `width="24"` and `height="24"`.

6. **defaults to currentColor** ‚Äî render without color prop, verify SVG has `fill="currentColor"`.

7. **sets displayName** ‚Äî call `createIcon('...', 'RiTestIcon')`, verify the returned component's `displayName` is `'RiTestIcon'`.

8. **is memoized** ‚Äî call `createIcon` with same args, render twice with same props, verify React.memo is applied (check `.$$typeof` or `displayName` includes `Memo` wrapper).

1. **RiArrowRightLine renders** ‚Äî render `<RiArrowRightLine />`, verify SVG is in the DOM.
