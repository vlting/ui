## CSS Variable Convention
Components use `var(--stl-{token}, {fallback})` pattern. The CSS var names from the STL system:
- Colors: `--stl-background`, `--stl-foreground`, `--stl-color1` through `--stl-color12`, `--stl-border`, `--stl-surface2`, `--stl-surface3`
- Shadows: `--stl-shadow-sm`, `--stl-shadow-md`, `--stl-shadow-lg`
- Space: `--stl-space0` through `--stl-space16`
- Size: `--stl-size0` through `--stl-size16`
- Radius: `--stl-radius0` through `--stl-radius12`
