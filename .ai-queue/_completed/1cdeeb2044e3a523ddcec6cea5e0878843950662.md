## Instructions

### Dashboard01 — Full Dashboard with Sidebar, Charts, Data Table
A complete dashboard layout combining a sidebar for navigation, chart visualizations, and a data table.

**Props:** `Dashboard01Props` with:
- `sidebarGroups: NavGroup[]` — sidebar navigation
- `sidebarHeader?: ReactNode`
- `sidebarFooter?: ReactNode`
- `title?: string` (default: "Dashboard")
- `description?: string`
- `chartConfig?: ChartConfig` — config for the chart (series labels/colors)
- `chartData?: Record<string, unknown>[]` — data for the charts
- `chartXAxisKey?: string` (default: "month")
- `tableData?: Record<string, unknown>[]` — data for the data table
- `tableColumns?: ColumnDef<Record<string, unknown>, unknown>[]` — column definitions for the table
- `children?: ReactNode` — optional extra content

**Layout:**
- Outer container with flexDirection="row" and flex={1}
- Left: Sidebar.Root with sidebarGroups (using SidebarNavGroup), header, footer
- Right: Main content area with flex={1}, padding
  - Top: Header area with title (Text, heading style) and description
  - Middle: Chart section — wrapped in Card.Root with Card.Header("Overview") and Card.Content containing a Chart with BarChart. Only render if chartConfig and chartData are provided.
  - Bottom: Data table section — wrapped in Card.Root with Card.Header("Recent Data") and Card.Content containing DataTable. Only render if tableData and tableColumns are provided.
  - If children provided, render after table

**Implementation notes:**
- Import ColumnDef type from @tanstack/react-table for the tableColumns prop type
- Chart uses BarChart variant by default with showGrid, showTooltip, showXAxis, showYAxis
- DataTable uses enablePagination={true} with default pageSize
- The sidebar uses the standard block pattern (SidebarNavGroup helpers)
