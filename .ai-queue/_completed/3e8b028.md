# Add missing complex components for shadcn/ui parity

## Goal

Add the more complex components that shadcn/ui provides. These involve richer state management, keyboard navigation, and/or composition of multiple sub-components.

## Components to Add

### 1. Calendar (`packages/components/Calendar/Calendar.tsx` — NEW dir)

Standalone month calendar for date selection (our DatePicker has an internal calendar, but shadcn exposes Calendar as its own component).
- Extract and expose the calendar grid from our existing DatePicker internals, OR build a standalone calendar component
- Props: `selected`, `onSelect`, `month`, `onMonthChange`, `disabled`, `minDate`, `maxDate`, `locale`, `mode` ('single' | 'range' | 'multiple')
- Features: month navigation, today highlighting, disabled dates, range selection highlighting
- Compound: `Calendar.Root`, `Calendar.Header`, `Calendar.Grid`, `Calendar.Day`
- Style: grid layout, day cells with hover/focus/selected states

### 2. Combobox (`packages/components/Combobox/Combobox.tsx` — NEW dir)

Searchable select (autocomplete dropdown).
- Builds on `@tamagui/popover` for the dropdown + Input for the search field
- Props: `options` (array of {value, label}), `value`, `onValueChange`, `placeholder`, `disabled`, `searchPlaceholder`, `emptyMessage`
- Features: type-to-filter, keyboard navigation (arrow keys, Enter, Escape), clear button
- Compound: `Combobox.Root`, `Combobox.Input`, `Combobox.Content`, `Combobox.Item`, `Combobox.Empty`
- Accessible: `role="combobox"`, `aria-expanded`, `aria-activedescendant`, listbox pattern

### 3. Command (`packages/components/Command/Command.tsx` — NEW dir)

Command palette / spotlight search (like Cmd+K).
- A searchable command list, typically shown in a Dialog
- Props: `open`, `onOpenChange`, `filter` (custom filter function), `loop` (keyboard navigation loops)
- Compound: `Command.Root`, `Command.Input`, `Command.List`, `Command.Empty`, `Command.Group`, `Command.Item`, `Command.Separator`, `Command.Loading`, `Command.Dialog` (wraps in Dialog)
- Features: search filtering, grouped items, keyboard navigation, item selection callback
- Accessible: `role="listbox"`, `aria-label`, keyboard-only operation
- Note: shadcn uses `cmdk` library. We can implement our own or keep it simple.

## Key Patterns

For ALL new components:
1. Use existing `@tamagui/` packages where available (menu, popover, sheet, etc.)
2. `styled()` wrap Tamagui components with our theme tokens
3. Proper ARIA roles and keyboard navigation
4. Export TypeScript types for all props
5. Compound component pattern with `withStaticProperties` or plain object
