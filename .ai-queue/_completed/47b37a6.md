### Pattern to Follow

1. Add an entry to `playgroundConfigs` with `component` name and `controls` array

3. Add a case to `generateCode()` that produces the code string

### Import Pattern
Add new component imports at the top using the AnyFC cast pattern:
```tsx

```

For compound components:
```tsx
import { Alert as _Alert } from '@vlting/ui'
const Alert = _Alert as AnyFC & { Root: AnyFC; Title: AnyFC; Description: AnyFC }
```
