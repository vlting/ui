### 2. Mobile responsive improvements
- **Docs layout** (`app/docs/layout.tsx`): Add a mobile menu button that toggles the sidebar. On mobile (<md), the sidebar should be a slide-over or sheet that appears when the menu button is clicked.

  - Close on navigation or backdrop click
  - Use CSS transitions for the slide-in effect
- **Site header** (`components/site-header.tsx`): Ensure the mobile layout works well — the nav is already hidden on mobile, but consider adding a mobile menu or at minimum making sure all content fits within narrow screens
