# Task: Fix StlProvider — rAF debounce, FOUC prevention script, Object.freeze exports

## Context
The StlProvider has a 50ms setTimeout for system color mode sync that causes perceptible flashes. We also need a FOUC prevention script export and Object.freeze on token exports.

## Requirements

### 1. Replace 50ms setTimeout with requestAnimationFrame

In `packages/stl-react/src/providers/StlProvider.tsx`:

Find the `useEffect` around line 90 that does:
```typescript
systemColorTimer.current = setTimeout(() => {
  setColorMode(systemColorMode)
}, 50)
```

Replace with:
```typescript
const rafId = requestAnimationFrame(() => {
  setColorMode(systemColorMode)
})
return () => cancelAnimationFrame(rafId)
```

This syncs with the next paint frame instead of an arbitrary 50ms delay.
