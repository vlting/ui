### Sidebar11 — Collapsible File Tree
A sidebar showing a file/folder tree structure with expand/collapse for directories.

**New types needed in `_shared.tsx`:**
```typescript
export interface FileTreeItem {
  name: string
  icon?: ReactNode
  type: 'file' | 'folder'
  children?: FileTreeItem[]
  onPress?: () => void
  active?: boolean
}
```
Add this type export to `_shared.tsx`. Also add a `FileTreeNode` helper component that recursively renders the tree.

**Props:** `Sidebar11Props` with:
- `tree: FileTreeItem[]` — the file tree data
- `header?: ReactNode`
- `footer?: ReactNode`
- Standard sidebar props (collapsible, side, variant, defaultOpen, open, onOpenChange, width, collapsedWidth)

**Layout:**
- Sidebar.Root with standard props
- Content area renders the file tree recursively
- Folders use Collapsible (expand/collapse with arrow indicator)
- Files are leaf items (clickable, show icon + name)
- Depth-based indentation using paddingLeft (similar to NestedNavItem pattern)
- File icon defaults to a simple text icon (📄) if none provided, folder defaults to 📁/📂

Create: `packages/blocks/sidebar/Sidebar11.tsx`, `packages/blocks/sidebar/Sidebar11.spec.md`, `packages/blocks/sidebar/Sidebar11.test.tsx`

### Sidebar12 — Sidebar with Calendar
A sidebar that includes a calendar date picker in the sidebar content, along with navigation items. Useful for date-based navigation (e.g., scheduling apps).

**Props:** `Sidebar12Props extends SidebarBlockProps` plus:
- `selectedDate?: Date`
- `onDateSelect?: (date: Date) => void`
- `calendarPosition?: 'top' | 'bottom'` (default: "top" — calendar above navigation)

**Layout:**
- Sidebar.Root with standard props
- If calendarPosition is "top": Calendar.Root first, then Separator, then navigation groups
- If calendarPosition is "bottom": navigation groups first, then Separator, then Calendar.Root
- Calendar uses mode="single", selected={selectedDate}, onSelect={onDateSelect}
- Navigation groups use the standard SidebarNavGroup helper

Create: `packages/blocks/sidebar/Sidebar12.tsx`, `packages/blocks/sidebar/Sidebar12.spec.md`, `packages/blocks/sidebar/Sidebar12.test.tsx`
