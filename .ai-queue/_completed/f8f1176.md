#### Quick Start
- Code example showing Provider setup with a brand:
```tsx
import { Provider, createBrandConfig, defaultBrand } from '@vlting/ui'
import { createTamagui } from 'tamagui'

const config = createTamagui(createBrandConfig(defaultBrand))

function App() {
  return <Provider config={config}>{children}</Provider>
}
```

#### Token Reference
- **Size tokens**: 0–16 scale + semantic tokens (sidebar, drawer, dialog, menu)
- **Space tokens**: 0–16 scale + negative values
- **Radius tokens**: 0–12 scale + `full` (9999)
- **Color tokens**: Tamagui semantic color tokens ($color, $background, $borderColor, etc.)
- **zIndex tokens**: 0–7 scale
