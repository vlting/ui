# Fix DropdownMenu and NavigationMenu Keyboard Navigation

## Objective

DropdownMenu and NavigationMenu are missing keyboard navigation — arrow key traversal, Escape to close, and Enter/Space to activate items. This violates WCAG 2.1.1 (keyboard operability). Fix both components to follow WAI-ARIA APG menu patterns.

## Context

Both components are custom implementations (not using Tamagui menu primitives). They have `role="menu"`, `aria-expanded`, and item ARIA attributes, but no keyboard event handling for menu navigation.

## Instructions

### 1. DropdownMenu (`packages/components/DropdownMenu/DropdownMenu.tsx`)

Add keyboard navigation to the menu content:
- **Arrow Down / Arrow Up**: Move focus between `role="menuitem"` items
- **Enter / Space**: Activate the focused item
- **Escape**: Close the menu and return focus to trigger
- **Home / End**: Move focus to first / last item
- **Tab**: Close menu (standard menu behavior — menus trap focus within)

Use the existing `useKeyboardNavigation` hook from `packages/hooks/` if applicable, or implement a lightweight roving tabindex pattern.

### 2. NavigationMenu (`packages/components/NavigationMenu/NavigationMenu.tsx`)

Add keyboard navigation:
- **Arrow Left / Arrow Right**: Move between top-level navigation triggers
- **Arrow Down**: Open submenu (if trigger has dropdown)
- **Enter / Space**: Activate link or open dropdown
- **Escape**: Close open submenu and return focus to trigger

### 3. Update Specs

Update `DropdownMenu.spec.md` and `NavigationMenu.spec.md` to reflect the new keyboard behavior in the Behavior > Keyboard section.

### 4. Update Tests

Update or add tests in `DropdownMenu.test.tsx` and `NavigationMenu.test.tsx` to verify keyboard navigation (at minimum, verify the event handlers are attached — full JSDOM keyboard simulation may be limited).

### 5. Verify

- Run tests: `npx jest packages/components/DropdownMenu/ packages/components/NavigationMenu/`
- Run type check: `npx tsc --noEmit`

## Scope

- **Modifies**: `packages/components/DropdownMenu/DropdownMenu.spec.md`, `packages/components/NavigationMenu/NavigationMenu.spec.md`
- **Modifies**: `packages/components/DropdownMenu/DropdownMenu.test.tsx`, `packages/components/NavigationMenu/NavigationMenu.test.tsx`
